CHOpt binaries go in "CHOpt" folder
https://github.com/GenericMadScientist/CHOpt/releases/
Download "CHOpt.CLI.VERSION.x64.Linux.zip"

Install FFmpeg separately if you want to skip this step.

FFmpeg binaries go in "FFmpeg" folder
https://github.com/BtbN/FFmpeg-Builds/releases/tag/latest
Download "ffmpeg-master-latest-linux64-lgpl.tar.xz"

Tipically, FFmpeg binaries are in "bin" folder, this folder must be present in the "FFmpeg" folder.
